# GUI Notes App 📝

A desktop notes-taking app using Python Tkinter.

## Features
- Multiline note input
- Auto timestamp on save
- View all notes
- Clear all notes with confirmation
- Error handling included

## Technologies
- Python
- Tkinter
- Text file I/O

## Usage
Run the script:
```bash
python gui_notes_app.py
